# Bombardiers — Gold Master

Bestanden:

- `index.html` — structuur van het spel
- `style.css` — vormgeving
- `game.js` — gameplay, fysica, AI, shield, muziek en effecten

## Starten

Open `index.html` in een moderne browser.

Voor lokaal ontwikkelen kun je ook vanuit deze map uitvoeren:

```bash
python3 -m http.server 8000
```

Open daarna `http://localhost:8000`.

Bewaar deze versie als vaste Gold Master, bijvoorbeeld met een Git-tag `gold-master-v1`.
